# README

Vue + Vuex + VueRouter + VueMeta Webpack starter-kit.

Used *.pug and *.scss files for vue-components

-----

# Run

* `npm start` - run for dev
* `npm run build` - run for production

or

* `yarn install`
* `bower install`
* `npm start`

# Files and folders

* src/js/app.js - Vue-app and router.
* src/js/components/ - Vue-components: *.js+*.pug+*.scss.
* src/js/store/ - Vuex store.
* src/js/services/ - services for api, meta, etc.
* src/js/config - settings
